About the Project

The Building & Blueprint Analyzer is a web-based deep learning project designed to simplify building analysis tasks. It offers two specialized tools: a Floor Counter that identifies the number of floors in building images, and a Room Categorization Tool that analyzes blueprints to label rooms like kitchen, bedroom, etc.

Our mission is to provide a user-friendly platform for architects, civil engineers, and urban developers, allowing them to upload building images or blueprints and receive precise results quickly. The system uses advanced machine learning models for analyzing structural features and outputs the results in an easy-to-download PDF format for further use.
